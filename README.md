
# Accounting App REST API

Simple secure accounting API allowing users to add their income and expenses.

## Tech Stack
- NodeJS
- NestJS (With Express.js)
- PostgreSQL
- TypeORM
- Swagger


## Demo
[https://cb-accounting-app-api.herokuapp.com/](https://cb-accounting-app-api.herokuapp.com/)


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
